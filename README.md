# Dragon Drop

## Lancement du projet

### Depuis le dossier front, exécuter les commandes suivantes:

`npm install`
puis:
`npx serve -l 8000`

### Depuis le dossier back, exécuter les commandes suivantes:

`npm install`
puis:
`npm run start`
